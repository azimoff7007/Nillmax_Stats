# Sales Dashboard 2024–2026

Расширенный HTML-дашборд для анализа продаж, прибыли, оборота и остатков товаров.

## Возможности

* Анализ продаж по годам
* Анализ прибыли по категориям
* Графики динамики продаж
* Остатки товаров
* Светлая и темная тема
* Интерактивные графики на Chart.js
* Полностью адаптивный дизайн

## Технологии

* HTML5
* CSS3
* JavaScript
* Chart.js
* Vercel

## Публикация

Проект автоматически деплоится через GitHub + Vercel.
